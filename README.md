US IPO Alert Automation

This project is an automated workflow that monitors same-day U.S. IPOs using the Nasdaq IPO calendar API. It filters IPOs with an IPO offering value above $200 million and sends a daily email alert.

The automation is designed to run every day at 9:00 AM Dubai time.
